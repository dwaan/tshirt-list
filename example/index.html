T-shirt List

T-shirt List is listing implementation. It read multi Array data and convert it into working listing complete with filtering and some more actions.

T-shirt List only required jQuery for basic fucntionality, but some feature required T-shirt Scroll. If you use bower, do bower update for easy instalation.

Data

We're using this data for example

Config

["check", "find", "side", "value"]

Config explanation:

  • check: The content will be displayed as check element, use eather 0 or 1 to choose between checked or unchecked.
  • find: The filtering function will use this content as a filtering data.
  • side: The content will be aligned to the right.
  • value: The content will be put as a data-value attribute and will be hidden.
  • input: The content will displayed as value of input element.
  • callback (index, data): Define a callback function in here, to further stylized with your own result.

We put config as a class name, but on special spesific term, we display it differently. Only one config for each element, so it can't be mixed.

Basic Listing

A basic listing, with all predefined parameters except the config and data.

$("#list").list({
	config: config,
	data: data
});

Listing with Highlighted result

Listing with the result highlighted. On very large data, this might resulting some slow down. The text will be inside strong tags.

$("#list").list({
	config: config,
	data: data,
	highlight: true
});

Listing with on click callback

Listing with on click callback on the row.

$("#list").list({
	config: config,
	data: data,
	onClick: function (element) {
		// Do something
	}
});

Result:

Listing with multiple select

Sometimes you need to do a multiple select on the listing.

$("#list").list({
	config: config,
	data: data,
	multipleSelect: true
});
